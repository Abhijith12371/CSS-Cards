Python History

Python was created by Guido van Rossum and first released in 1991. It has since evolved through many versions, each bringing new features and improvements. Python's simplicity and readability have made it one of the most popular programming languages in the world.

Python Interactive Mode and Script Mode

Interactive mode allows you to type Python commands and see the results immediately. It's useful for quick tests and debugging.

>>> print("Hello, World!")
Hello, World!

In script mode, you write your code in a file with a `.py` extension and run it using the Python interpreter.

# script.py
print("Hello, World!")

# Run the script
$ python script.py

IDLE File Saving and Execution

Python's Integrated Development and Learning Environment (IDLE) is a handy tool to write, save, and run Python scripts.

  1. Open IDLE from your applications.
  2. Write your Python code in the editor window.
  3. Save the file with a .py extension (e.g., script.py).
  4. Run the script by pressing F5 or selecting Run > Run Module from the menu.

String Indexing and Slicing

Strings can be indexed (subscripted), with the first character having index 0.

message = "Hello, World!"

print(message[0])    # 'H'
print(message[7])    # 'W'

# Slicing
print(message[7:12])  # 'World'
print(message[:5])    # 'Hello'
print(message[7:])    # 'World!'

Python Lists

Lists are used to store multiple items in a single variable.

Methods that Modify the List In-Place

fruits = ["apple", "banana", "cherry"]

# append(x): Adds element x to the end of the list.
fruits.append("date")
print(fruits)  # ['apple', 'banana', 'cherry', 'date']

# extend(iterable): Extends the list by appending all elements from the iterable.
fruits.extend(["grape", "melon"])
print(fruits)  # ['apple', 'banana', 'cherry', 'date', 'grape', 'melon']

# insert(i, x): Inserts element x at index i.
fruits.insert(1, "orange")
print(fruits)  # ['apple', 'orange', 'banana', 'cherry', 'date', 'grape', 'melon']

# remove(x): Removes the first item from the list whose value is x.
fruits.remove("banana")
print(fruits)  # ['apple', 'orange', 'cherry', 'date', 'grape', 'melon']

# pop([i]): Removes and returns the item at index i. If i is not specified, removes and returns the last item in the list.
popped = fruits.pop(2)
print(fruits)  # ['apple', 'orange', 'date', 'grape', 'melon']
print(popped)  # cherry

# clear(): Removes all items from the list.
fruits.clear()
print(fruits)  # []

Methods that Return a Modified Copy of the List

fruits = ["apple", "banana", "cherry", "banana"]

# copy(): Returns a shallow copy of the list.
fruits_copy = fruits.copy()
print(fruits_copy)  # ['apple', 'banana', 'cherry', 'banana']

# count(x): Returns the number of times x appears in the list.
count_banana = fruits.count("banana")
print(count_banana)  # 2

# index(x[, start[, end]]): Returns the index of the first occurrence of x in the list (optional start and end indices).
index_banana = fruits.index("banana")
print(index_banana)  # 1

Additional Utility Methods

# len(): Returns the number of items in the list (not a method of list objects, but commonly used with lists).
print(len(fruits))  # 4

Python Dictionaries

Dictionaries store data in key-value pairs.

Methods for Dictionaries

student = {"name": "Alice", "age": 20}

# clear(): Removes all items from the dictionary.
student.clear()
print(student)  # {}

# copy(): Returns a shallow copy of the dictionary.
student = {"name": "Alice", "age": 20}
student_copy = student.copy()
print(student_copy)  # {'name': 'Alice', 'age': 20}

# get(key[, default]): Returns the value for key if key is in the dictionary, else default.
name = student.get("name", "Unknown")
print(name)  # Alice

# items(): Returns a view object that displays a list of a dictionary's key-value tuple pairs.
items = student.items()
print(items)  # dict_items([('name', 'Alice'), ('age', 20)])

# keys(): Returns a view object that displays a list of all the keys in the dictionary.
keys = student.keys()
print(keys)  # dict_keys(['name', 'age'])

# values(): Returns a view object that displays a list of all the values in the dictionary.
values = student.values()
print(values)  # dict_values(['Alice', 20])

# pop(key[, default]): Removes the item with the specified key and returns its value or default if key is not found.
age = student.pop("age")
print(age)  # 20

# popitem(): Removes and returns an arbitrary key-value pair from the dictionary.
item = student.popitem()
print(item)  # ('name', 'Alice')

# update(iterable): Updates the dictionary with the items from the iterable.
student.update({"grade": "A"})
print(student)  # {'grade': 'A'}

Python Sets

Sets are unordered collections of unique elements.

Methods for Sets

s = {"apple", "banana", "cherry"}

# add(x): Adds element x to the set.
s.add("date")
print(s)  # {'apple', 'banana', 'cherry', 'date'}

# remove(x): Removes the specified element from the set.
s.remove("banana")
print(s)  # {'apple', 'cherry', 'date'}

Python Sets

A set is a collection that is unordered and unindexed. It does not allow duplicate members.

s = {"apple", "banana", "cherry"}

# Accessing items
for item in s:
    print(item)

# Adding items
s.add("orange")
print(s)    # {'apple', 'cherry', 'orange', 'banana'}

Python Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is " + self.name)

# Create an object
p1 = Person("Alice", 20)
p1.greet()

Python OOP Concepts

Python supports object-oriented programming (OOP) with features like inheritance, encapsulation, and polymorphism.

Inheritance

class Student(Person):
    def __init__(self, name, age, major):
        super().__init__(name, age)
        self.major = major

    def study(self):
        print(self.name + " is studying " + self.major)

# Create an object
s1 = Student("Bob", 21, "Computer Science")
s1.study()

Encapsulation

class Car:
    def __init__(self, brand):
        self.__brand = brand

    def get_brand(self):
        return self.__brand

    def set_brand(self, brand):
        self.__brand = brand

# Create an object
car = Car("Toyota")
print(car.get_brand())    # 'Toyota'
car.set_brand("Honda")
print(car.get_brand())    # 'Honda'

Polymorphism

class Animal:
    def sound(self):
        pass

class Dog(Animal):
    def sound(self):
        print("Bark")

class Cat(Animal):
    def sound(self):
        print("Meow")

# Create objects
dog = Dog()
cat = Cat()

dog.sound()    # 'Bark'
cat.sound()    # 'Meow'

Python More Concepts

Python provides several additional features and functionalities.

Lambda Functions

f = lambda x: x**2
print(f(5))    # 25

List Comprehension

numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
print(squares)    # [1, 4, 9, 16, 25]

Map, Filter, and Reduce

numbers = [1, 2, 3, 4, 5]

# Map
doubled = list(map(lambda x: x*2, numbers))
print(doubled)    # [2, 4, 6, 8, 10]

# Filter
evens = list(filter(lambda x: x%2 == 0, numbers))
print(evens)    # [2, 4]

# Reduce
from functools import reduce
sum_all = reduce(lambda x, y: x+y, numbers)
print(sum_all)    # 15

Decorators

def my_decorator(func):
    def wrapper():
        print("Something is happening before the function is called.")
        func()
        print("Something is happening after the function is called.")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
